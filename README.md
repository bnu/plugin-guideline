# XE3 테마, 게시판 스킨 Guideline 플러그인

이 플러그인은 XE3의 테마 및 게시판 스킨을 개발할 때 필요한 필수 구성을 담고있습니다.

테마, 스킨의 필수 구성을 갖추고 있으며, 화면에 출력할 수 있는 데이터를 다루는 방법의 예시와 설명을 볼 수 있습니다.
